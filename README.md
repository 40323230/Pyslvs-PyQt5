Pyslvs
===

A GUI-based tool use Ubuntu version to solve 2D and 3D linkage subject.

Input formate
---

1. CSV table sheet

1. Solvespace formate (`.slvs`)

Output
---

Including Path coordinate and file info.

1. CSV table sheet

1. Solvespace formate (`.slvs`)

1. PNG Image

1. SLT formate

1. Python 3 script
