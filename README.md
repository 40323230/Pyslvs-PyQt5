Pyslvs(PySolveSpace)
===

A GUI-based tool solving 2D linkage subject.

Compatible with Python 3.4 and above.

![](icons/cover.png)

How to use
---

Open GUI by Python:

<pre>
$python3 launch_pyslvs.py
</pre>

Or see help:

<pre>
$python3 launch_pyslvs.py --help
</pre>

Input formate
---

1. CSV table sheet

1. Solvespace formate (`.slvs`)

Output
---

Including Path coordinate and file info.

1. CSV table sheet

1. Solvespace formate (`.slvs`)

1. PNG Image

1. DXF formate

1. SLT formate

1. Python 3 script
